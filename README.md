Industrial **RS485 Modbus RTU → MQTT** gateway for legacy meters/sensors → cloud dashboards.  
**ESP32 | ₹2000 India | €25 DE | Hardware Jan 2026**

## Features
- Modbus Master (RTU/ASCII, 9600-115200 baud)
- Multi-slave polling (ID 1-247)
- MQTT JSON publish (Home Assistant/Node-RED/Grafana)
- OTA updates, Web Config
- Industrial DIN enclosure ready
